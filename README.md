## 🚀 Cara Menjalankan Lokal

### Prerequisites
- Web browser modern (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, dll)

### Langkah Instalasi
1. **Clone atau Download**
   ```bash
   git clone https://github.com/username/course-html.git
   cd course-html
   ```

2. **Buka di Browser**
   - Buka file `index.html` di browser
   - Atau gunakan live server di VS Code

3. **Mulai Belajar**
   - Pilih modul dari sidebar
   - Ikuti materi secara berurutan
   - Praktikkan contoh kode yang diberikan

## 🌐 Deploy ke GitHub Pages

### Langkah 1: Upload ke GitHub
1. Buat repository baru di GitHub
2. Upload semua file ke repository
3. Pastikan struktur folder sudah benar

### Langkah 2: Setup GitHub Pages
1. Masuk ke repository di GitHub
2. Klik **Settings** → **Pages**
3. Pilih **Source**: `Deploy from a branch`
4. Pilih **Branch**: `main` dan folder `/ (root)`
5. Klik **Save**

### Langkah 3: Akses Website
- Tunggu beberapa menit untuk deployment
- Akses di `https://username.github.io/repo-name`
- Website siap digunakan!

## 🎨 Customization

### Mengubah Tema
- Edit CSS variables di `style.css`
- Ubah warna primary, secondary, dll
- Sesuaikan dengan brand colors

### Menambah Modul
1. Buat file HTML baru di folder `modules/`
2. Update `moduleData` di `script.js`
3. Tambahkan konten sesuai struktur yang ada

### Mengubah Layout
- Edit CSS di `style.css`
- Sesuaikan grid, flexbox, atau layout lainnya
- Test responsiveness di berbagai ukuran layar
