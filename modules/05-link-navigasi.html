Modul 5: Link & Navigasi

Mengenal Link di HTML

Link (<a>) digunakan untuk membuat tautan antar halaman, ke website lain, atau ke bagian tertentu dalam halaman.

Contoh Link Dasar

<a href="https://www.google.com" target="_blank">Kunjungi Google</a>

Link Internal & Eksternal

  • Internal: Menuju halaman lain di website yang sama.
    <a href="about.html">Tentang Kami</a>
  • Eksternal: Menuju website lain.
    <a href="https://www.wikipedia.org" target="_blank">Wikipedia</a>

Anchor Link (Tautan ke Bagian Halaman)

<a href="#bagian2">Lompat ke Bagian 2</a>
...
<h2 id="bagian2">Bagian 2</h2>

Latihan

  1. Buat file link-navigasi.html.

  4. Buka di browser dan uji semua link.
