Modul 11: Best Practice & SEO Dasar

Best Practice Penulisan HTML

  • Gunakan struktur HTML yang benar dan rapi (indentasi, penamaan tag jelas).
  • Selalu gunakan <!DOCTYPE html> dan lang pada tag <html>.
  • Gunakan semantic HTML untuk struktur yang jelas.
  • Tambahkan alt pada gambar untuk aksesibilitas.
  • Validasi kode HTML di W3C Validator.

Dasar SEO (Search Engine Optimization)

  • Gunakan tag <title> yang deskriptif dan unik di setiap halaman.
  • Tambahkan <meta name="description"> untuk ringkasan halaman.
  • Gunakan heading (<h1>, <h2>, dst) secara berurutan.
  • Gunakan URL yang ramah (tanpa karakter aneh).
  • Optimalkan gambar (ukuran, alt deskriptif).
  • Gunakan <a> dengan teks link yang jelas.

Contoh Meta Tag SEO

<head>
  <title>Belajar HTML Lengkap</title>
  <meta name="description" content="Kursus HTML terlengkap dari dasar hingga mahir.">
  <meta name="keywords" content="html, kursus, belajar, web">
  <meta name="author" content="Nama Anda">
</head>

Ilustrasi SEO

Latihan

  1. Buat file seo.html.
  2. Tambahkan meta tag SEO di <head> dan struktur heading yang benar.
  3. Validasi kode HTML Anda di W3C Validator.
