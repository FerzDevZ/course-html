        Modul 12: HTML Lanjutan

        Pelajari fitur-fitur lanjutan HTML5 untuk website modern

        Meta Tag & Favicon

        Meta tag digunakan untuk memberikan informasi tambahan pada browser dan mesin pencari. Favicon adalah ikon kecil yang muncul di tab browser.

        Contoh Meta Tag & Favicon:

        <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Belajar HTML Lanjutan">
  <link rel="icon" href="favicon.ico">
  <title>HTML Lanjutan</title>
</head>

        Fitur Lanjutan HTML5

        • Audio & Video: Menyisipkan media tanpa plugin eksternal
        • Canvas: Menggambar grafik dan animasi
        • LocalStorage: Menyimpan data di browser
        • Geolocation: Mendapatkan lokasi pengguna
        • Drag & Drop: Interaksi seret dan lepas

        Contoh Audio & Video:

        <audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls width="320" height="240">
  <source src="video.mp4" type="video/mp4">
</video>

        Latihan

        1. Tambahkan meta tag dan favicon pada file HTML kamu.
        2. Coba sisipkan audio dan video ke halaman web.
        3. Eksplorasi fitur localStorage dan canvas.
