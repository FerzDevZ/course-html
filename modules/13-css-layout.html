        Modul 13: CSS Layout

        Pelajari teknik layout modern dengan CSS Flexbox dan Grid

        CSS Layout Modern

        CSS Layout adalah teknik untuk mengatur posisi dan susunan elemen-elemen dalam halaman web. Ada beberapa metode layout yang dapat digunakan:

        Analogi & Penjelasan

        Layout = Puzzle

        Layout CSS seperti menyusun puzzle. Setiap elemen adalah kepingan puzzle yang harus ditempatkan dengan tepat untuk membentuk gambar yang utuh.

        Layout = Arsitektur

        Layout seperti merancang denah rumah. Kita menentukan di mana dinding, ruangan, dan pintu akan ditempatkan untuk menciptakan struktur yang fungsional.

        CSS Flexbox

        Flexbox adalah model layout satu dimensi yang dirancang untuk mengatur elemen dalam satu arah (baris atau kolom).

        Contoh Flexbox:

        .container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-direction: row;
}

.item {
  flex: 1;
  margin: 10px;
}

        Property Flexbox Utama:

        • display: flex - Mengaktifkan flexbox
        • justify-content - Mengatur alignment horizontal
        • align-items - Mengatur alignment vertical
        • flex-direction - Mengatur arah flex (row/column)
        • flex-wrap - Mengatur wrapping elemen

        CSS Grid

        CSS Grid adalah model layout dua dimensi yang memungkinkan kita membuat layout kompleks dengan baris dan kolom.

        Contoh CSS Grid:

        .grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto auto;
  gap: 20px;
}

.grid-item {
  padding: 20px;
  background: #f0f0f0;
}

        Property Grid Utama:

        • display: grid - Mengaktifkan grid
        • grid-template-columns - Mendefinisikan kolom
        • grid-template-rows - Mendefinisikan baris
        • gap - Jarak antar grid item
        • grid-area - Menentukan area grid

        CSS Positioning

        Positioning digunakan untuk mengatur posisi elemen relatif terhadap elemen lain atau viewport.

        Jenis Positioning:

        .relative {
  position: relative;
  top: 10px;
  left: 20px;
}

.absolute {
  position: absolute;
  top: 0;
  right: 0;
}

.fixed {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

.sticky {
  position: sticky;
  top: 0;
}

        Latihan Praktis

        1. Buat layout header, sidebar, dan content menggunakan flexbox
        2. Coba buat grid 3x3 dengan CSS Grid
        3. Eksperimen dengan positioning untuk membuat floating button
        4. Buat responsive layout yang berubah dari desktop ke mobile
