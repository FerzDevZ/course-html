26. Proyek: Landing Page

Buat landing page modern dan responsif dengan HTML & CSS

Overview Proyek

Dalam proyek ini, kita akan membuat landing page modern untuk sebuah startup teknologi. Landing page ini akan mencakup semua elemen penting seperti hero section, fitur-fitur, testimonial, dan call-to-action yang efektif.

Tujuan Pembelajaran

  • Menguasai layout modern dengan Flexbox dan CSS Grid
  • Membuat desain responsif untuk berbagai ukuran layar
  • Mengimplementasikan animasi dan transisi CSS
  • Mengoptimalkan UX dengan navigasi yang intuitif
  • Menggunakan CSS variables untuk konsistensi desain

Struktur Landing Page

Komponen yang akan dibuat:

  1. Header/Navigation - Menu navigasi responsif
  2. Hero Section - Bagian utama dengan headline dan CTA
  3. Features Section - Fitur-fitur produk
  4. About Section - Informasi tentang perusahaan
  5. Testimonials - Ulasan pelanggan
  6. Pricing Section - Paket harga
  7. Contact Section - Formulir kontak
  8. Footer - Informasi tambahan dan social links

Latihan & Implementasi

Latihan 1: Setup Proyek

Buat struktur folder dan file untuk proyek landing page:

  • Buat folder landing-page
  • Buat file index.html
  • Buat file style.css
  • Buat file script.js
  • Buat folder images untuk gambar

Latihan 2: Implementasi Hero Section

Implementasikan hero section dengan:

  • Background gradient yang menarik
  • Typography yang jelas dan readable
  • Call-to-action buttons
  • Responsive design
Tips Implementasi
  • Gunakan CSS Grid untuk layout
  • Tambahkan animasi hover pada buttons
  • Optimalkan untuk mobile-first design
  • Gunakan placeholder image atau icon

Latihan 3: Features Section

Buat features section dengan:

  • Grid layout untuk feature cards
  • Icon dari Font Awesome
  • Hover effects
  • Consistent spacing

Tips & Best Practices

Mobile-First Design

Mulai dengan design mobile, kemudian scale up untuk desktop.

Optimasi Performance

Gunakan CSS yang efisien dan optimalkan gambar.

Accessibility

Pastikan website dapat diakses oleh semua pengguna.

SEO Friendly

Gunakan semantic HTML dan meta tags yang tepat.
