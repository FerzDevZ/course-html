        Modul 17: CSS Box Model

        Pelajari konsep dasar box model dalam CSS

        Apa itu CSS Box Model?

        CSS Box Model adalah konsep fundamental dalam CSS yang menjelaskan bagaimana setiap elemen HTML diperlakukan sebagai kotak dengan content, padding, border, dan margin.

        Analogi & Penjelasan

        Box Model = Kotak Hadiah

        Seperti kotak hadiah yang memiliki hadiah di dalam (content), bungkus kertas (padding), pita (border), dan ruang antar kotak (margin).

        Box Model = Rumah

        Seperti rumah dengan ruangan dalam (content), dinding dalam (padding), dinding luar (border), dan halaman (margin).

        Komponen Box Model

        Topologi Box Model

        Content

        Area yang berisi teks, gambar, atau elemen lainnya

        Padding

        Ruang kosong di dalam border, antara content dan border

        Border

        Garis yang mengelilingi padding dan content

        Margin

        Ruang kosong di luar border, antara elemen

        Contoh CSS Box Model

        CSS Box Model Properties:

        .box {
  /* Content */
  width: 200px;
  height: 100px;
  
  /* Padding */
  padding: 20px;
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 10px;
  padding-left: 15px;
  
  /* Border */
  border: 2px solid #333;
  border-radius: 8px;
  
  /* Margin */
  margin: 10px;
  margin-top: 5px;
  margin-right: 10px;
  margin-bottom: 5px;
  margin-left: 10px;
}

        Box Sizing

        Property box-sizing menentukan bagaimana width dan height dihitung.

        Box Sizing Properties:

        /* Default - content-box */
.element {
  box-sizing: content-box;
  width: 200px; /* Hanya content */
  padding: 20px; /* Ditambah ke width */
  border: 2px solid black; /* Ditambah ke width */
  /* Total width = 200 + 40 + 4 = 244px */
}

/* Border-box */
.element {
  box-sizing: border-box;
  width: 200px; /* Termasuk padding dan border */
  padding: 20px;
  border: 2px solid black;
  /* Total width = 200px (fixed) */
}

        Latihan Praktis

        1. Buat elemen dengan padding, border, dan margin yang berbeda
        2. Eksperimen dengan box-sizing content-box vs border-box
        3. Buat layout dengan margin auto untuk centering
        4. Coba negative margin untuk efek overlap
