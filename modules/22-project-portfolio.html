        Modul Selesai

        Modul 22: Project Portfolio

        Buat portfolio website profesional dengan HTML dan CSS

        Portfolio Website

        Portfolio website adalah cara terbaik untuk menampilkan skill dan karya kamu kepada dunia. Website ini akan menjadi showcase profesional kamu.

        Analogi & Penjelasan

        Portfolio = Galeri Seni

        Seperti galeri seni yang menampilkan karya terbaik seniman, portfolio website menampilkan karya terbaik kamu sebagai developer.

        Portfolio = CV Digital

        Portfolio adalah CV digital yang interaktif dan menarik, lebih dari sekadar dokumen statis.

        Struktur Portfolio

        Komponen Portfolio

        Header

        Navigation, logo, dan hero section

        About

        Informasi pribadi dan skill

        Projects

        Showcase karya terbaik

        Contact

        Informasi kontak dan social media

        CSS Styling

        CSS untuk Portfolio:

        /* Reset dan Base Styles */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Arial', sans-serif;
  line-height: 1.6;
  color: #333;
}

/* Header Styles */
.header {
  position: fixed;
  top: 0;
  width: 100%;
  background: rgba(255, 255, 255, 0.95);
  backdrop-filter: blur(10px);
  z-index: 1000;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem 5%;
  max-width: 1200px;
  margin: 0 auto;
}

.nav-links {
  display: flex;
  list-style: none;
  gap: 2rem;
}

.nav-links a {
  text-decoration: none;
  color: #333;
  font-weight: 500;
  transition: color 0.3s;
}

.nav-links a:hover {
  color: #007bff;
}

/* Hero Section */
.hero {
  height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  text-align: center;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
}

.hero-content h1 {
  font-size: 3rem;
  margin-bottom: 1rem;
}

.cta-button {
  padding: 1rem 2rem;
  background: #007bff;
  color: white;
  border: none;
  border-radius: 5px;
  font-size: 1.1rem;
  cursor: pointer;
  transition: background 0.3s;
}

.cta-button:hover {
  background: #0056b3;
}

/* Project Grid */
.project-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
  padding: 2rem 0;
}

.project-card {
  background: white;
  border-radius: 10px;
  overflow: hidden;
  box-shadow: 0 5px 15px rgba(0,0,0,0.1);
  transition: transform 0.3s;
}

.project-card:hover {
  transform: translateY(-5px);
}

.project-card img {
  width: 100%;
  height: 200px;
  object-fit: cover;
}

/* Responsive Design */
@media (max-width: 768px) {
  .nav-links {
    display: none;
  }
  
  .hero-content h1 {
    font-size: 2rem;
  }
  
  .project-grid {
    grid-template-columns: 1fr;
  }
}

        Langkah-langkah Pembuatan

        1. Buat struktur HTML dengan semantic tags
        2. Desain layout dengan CSS Grid dan Flexbox
        3. Tambahkan responsive design untuk mobile
        4. Implementasikan smooth scrolling navigation
        5. Tambahkan hover effects dan animations
        6. Optimasi untuk SEO dan performance
