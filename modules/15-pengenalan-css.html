15. Pengenalan CSS

Pelajari dasar-dasar CSS untuk styling website

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa styling yang digunakan untuk mengatur tampilan dan format dokumen HTML. CSS memisahkan konten (HTML) dari presentasi (styling), sehingga website menjadi lebih mudah dikelola dan dipelihara.

Mengapa CSS Penting?

  • Konsistensi: Memastikan tampilan seragam di seluruh website
  • Maintainability: Mudah mengubah style tanpa mengubah HTML
  • Responsiveness: Website dapat menyesuaikan dengan berbagai ukuran layar
  • Performance: File CSS dapat di-cache browser

Cara Menambahkan CSS

1. Inline CSS

CSS ditulis langsung dalam atribut style elemen HTML:

<h1 style="color: blue; font-size: 24px;">Judul Biru</h1>

2. Internal CSS

CSS ditulis dalam tag <style> di bagian <head>:

<head>
  <style>
    h1 {
      color: blue;
      font-size: 24px;
    }
  </style>
</head>

3. External CSS (Direkomendasikan)

CSS ditulis dalam file terpisah dan di-link ke HTML:

<head>
  <link rel="stylesheet" href="style.css">
</head>

Sintaks CSS

CSS terdiri dari selector dan declaration block:

Struktur Dasar CSS:

selector {
  property: value;
  property: value;
}

Contoh Praktis:

h1 {
  color: #2563eb;
  font-size: 2rem;
  font-weight: bold;
  text-align: center;
  margin-bottom: 1rem;
}

Hasil:

Judul dengan CSS

Jenis-jenis Selector

Element Selector

p {
  color: red;
}

Mengatur semua elemen <p>

Class Selector

.highlight {
  background: yellow;
}

Mengatur elemen dengan class="highlight"

ID Selector

#header {
  background: blue;
}

Mengatur elemen dengan id="header"

Descendant Selector

div p {
  margin: 10px;
}

Mengatur <p> yang ada di dalam <div>

Property CSS Dasar

Property Warna dan Teks:

Property Fungsi Contoh
color Warna teks color: red;
background-color Warna latar belakang background-color: #f0f0f0;
font-size Ukuran font font-size: 16px;
font-family Jenis font font-family: Arial, sans-serif;
text-align Perataan teks text-align: center;

Latihan Praktis

Latihan 1: Styling Paragraf

Buat file CSS untuk mengatur tampilan paragraf dengan ketentuan:

  • Warna teks: biru (#2563eb)
  • Ukuran font: 18px
  • Jarak antar baris: 1.6
  • Margin bawah: 20px
Lihat Solusi 
p {
  color: #2563eb;
  font-size: 18px;
  line-height: 1.6;
  margin-bottom: 20px;
}

Latihan 2: Styling Heading

Buat style untuk heading h1 dengan ketentuan:

  • Warna: hijau (#10b981)
  • Ukuran: 2.5rem
  • Font weight: bold
  • Perataan: center
Lihat Solusi 
h1 {
  color: #10b981;
  font-size: 2.5rem;
  font-weight: bold;
  text-align: center;
}

Best Practices

Gunakan External CSS

Pisahkan CSS dalam file terpisah untuk maintainability yang lebih baik.

Gunakan Nama yang Deskriptif

Beri nama class dan ID yang jelas dan mudah dipahami.

Gunakan CSS Variables

Manfaatkan CSS custom properties untuk konsistensi warna dan ukuran.

Optimalkan Selector

Gunakan selector yang spesifik dan efisien untuk performa yang lebih baik.
