16. CSS Selector

Pelajari cara memilih dan mengatur elemen HTML dengan CSS Selector

Apa itu CSS Selector?

CSS Selector adalah cara untuk memilih elemen HTML yang ingin kita style. Seperti seorang detektif yang mencari target tertentu, selector membantu kita menemukan dan mengatur elemen yang tepat.

Mengapa Selector Penting?

  • Presisi: Memilih elemen yang tepat tanpa mempengaruhi yang lain
  • Efisiensi: Mengurangi jumlah kode CSS yang diperlukan
  • Maintainability: Mudah mengubah style untuk elemen tertentu
  • Flexibility: Berbagai cara untuk mencapai hasil yang sama

Jenis-jenis Selector

Element Selector

p {
  color: red;
}

Mengatur semua elemen <p>

Contoh paragraf dengan warna merah

Class Selector

.highlight {
  background: yellow;
}

Mengatur elemen dengan class="highlight"

Contoh dengan class highlight

ID Selector

#header {
  background: blue;
}

Mengatur elemen dengan id="header"

Header dengan ID

Descendant Selector

div p {
  margin: 10px;
}

Mengatur <p> yang ada di dalam <div>

Paragraf di dalam div

Child Selector

div > p {
  font-weight: bold;
}

Mengatur <p> yang langsung di dalam <div>

Pseudo-class Selector

Pseudo-class yang Sering Digunakan:

Selector Fungsi Contoh
:hover Ketika mouse di atas elemen button:hover { background: blue; }
:active Ketika elemen sedang diklik button:active { transform: scale(0.95); }
:focus Ketika elemen mendapat fokus input:focus { border-color: blue; }
:first-child Elemen pertama dari parent li:first-child { font-weight: bold; }
:last-child Elemen terakhir dari parent li:last-child { border-bottom: none; }
:nth-child(n) Elemen ke-n dari parent li:nth-child(odd) { background: #f0f0f0; }

Latihan Praktis

Latihan 1: Selector Dasar

Buat CSS untuk mengatur elemen berikut:

  • Semua heading h2 berwarna biru
  • Elemen dengan class "important" berwarna merah
  • Elemen dengan id "main-title" berukuran 2rem
Lihat Solusi 
h2 {
  color: blue;
}

.important {
  color: red;
}

#main-title {
  font-size: 2rem;
}

Latihan 2: Selector Kompleks

Buat CSS untuk:

  • Paragraf di dalam div dengan class "container"
  • Button yang langsung setelah heading h1
  • Elemen li yang ganjil dalam ul
Lihat Solusi 
.container p {
  line-height: 1.6;
}

h1 + button {
  margin-top: 1rem;
}

ul li:nth-child(odd) {
  background: #f0f0f0;
}

Best Practices

Gunakan Class untuk Reusability

Class lebih fleksibel dan dapat digunakan ulang di berbagai elemen.

Hindari ID untuk Styling

ID terlalu spesifik dan sulit untuk override. Gunakan untuk JavaScript.

Pahami Specificity

Ketahui urutan prioritas selector untuk menghindari konflik.

Gunakan Nama yang Deskriptif

Beri nama class yang jelas dan mudah dipahami.
