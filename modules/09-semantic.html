Modul 9: Semantic HTML

Apa itu Semantic HTML?

Semantic HTML adalah penggunaan tag HTML yang mendeskripsikan makna (arti) dari konten, bukan hanya tampilannya. Ini membantu SEO, aksesibilitas, dan struktur kode yang lebih baik.

Contoh Tag Semantic

  • <header>: Bagian atas halaman (judul, navigasi)
  • <nav>: Navigasi
  • <main>: Konten utama
  • <article>: Artikel mandiri
  • <section>: Bagian dari halaman
  • <aside>: Konten sampingan
  • <footer>: Bagian bawah halaman
  • <figure>, <figcaption>: Gambar & caption

Contoh Struktur Semantic HTML

<header>
  <h1>Judul Website</h1>
  <nav>...</nav>
</header>
<main>
  <article>
    <h2>Judul Artikel</h2>
    <section>...</section>
  </article>
  <aside>Sidebar</aside>
</main>
<footer>Copyright 2024</footer>

Ilustrasi Semantic HTML

Latihan

  1. Buat file semantic.html.
  2. Buat struktur halaman dengan <header>, <nav>, <main>, <footer>.
  3. Tambahkan <article>, <section>, dan <aside> sesuai kebutuhan.
  4. Buka di browser dan amati struktur HTML-nya.
