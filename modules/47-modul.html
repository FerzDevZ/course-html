        Modul 1: Pengenalan HTML

        Memulai perjalanan belajar HTML dari dasar

        Apa itu HTML?

        HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur, isi, dan elemen-elemen pada halaman web.

        Analogi & Penjelasan

        HTML = Kerangka Rumah

        HTML adalah seperti kerangka rumah. Ia menentukan di mana dinding, pintu, jendela, dan ruangan berada. Tanpa kerangka, rumah tidak bisa berdiri.

        HTML = Struktur Buku

        HTML seperti struktur buku dengan bab, sub-bab, paragraf, dan daftar isi. Ia mengorganisir konten agar mudah dibaca dan dipahami.

        Sejarah Singkat HTML

        • Diciptakan oleh Tim Berners-Lee pada tahun 1991
        • HTML berkembang dari versi 1.0 hingga HTML5 (saat ini)
        • HTML5 membawa banyak fitur modern seperti video, audio, canvas, dan semantic tag
        • Menjadi standar web yang dikelola oleh W3C (World Wide Web Consortium)

        Fungsi HTML

        • Membuat struktur dasar halaman web
        • Menampilkan teks, gambar, video, dan elemen multimedia
        • Menghubungkan antar halaman (hyperlink)
        • Menjadi fondasi sebelum menambahkan CSS & JavaScript
        • Membuat form untuk input data pengguna
        • Mengorganisir konten dengan heading, paragraf, dan list

        Contoh Kode HTML Sederhana

        Struktur HTML Dasar:

        <!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo, dunia!</h1>
    <p>Ini adalah paragraf pertama.</p>
    <p>Selamat datang di dunia HTML!</p>
  </body>
</html>

        Latihan Praktis

        1. Buka text editor (Notepad, VSCode, Sublime Text, dsb)
        2. Buat file baru dengan nama index.html
        3. Tulis kode HTML sederhana seperti contoh di atas
        4. Simpan file dan buka di browser (Chrome, Firefox, Safari)
        5. Lihat hasilnya dan coba ubah teks dalam tag <h1> dan <p>
