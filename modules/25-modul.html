        Modul 25: HTML & CSS Interview Questions

        Kumpulan soal interview HTML & CSS beserta tips dan pembahasan

        Kenapa Interview HTML & CSS Penting?

        Interview HTML & CSS sering menjadi tahap awal seleksi kerja web developer. Memahami tipe soal dan cara menjawabnya akan meningkatkan peluang lolos seleksi.

        Tips Sukses Interview

        • Pahami konsep dasar, bukan sekadar hafal sintaks
        • Latihan menjelaskan konsep dengan bahasa sendiri
        • Berikan contoh kode sederhana saat menjawab
        • Jangan ragu bertanya jika soal kurang jelas
        • Latihan live coding dan debugging

        Contoh Soal & Jawaban

        1. Apa itu semantic HTML? Berikan 3 contoh tag-nya.

        Jawaban: Semantic HTML adalah elemen HTML yang mendeskripsikan makna konten, bukan hanya tampilannya. Contoh: <header>, <nav>, <article>, <footer>.

        2. Apa perbedaan class dan id?

        Jawaban: class bisa dipakai banyak elemen, id hanya satu elemen per halaman. id unik, class untuk grup.

        3. Bagaimana cara membuat layout 2 kolom responsif dengan CSS?

        .container {
  display: flex;
  flex-wrap: wrap;
}
.col {
  flex: 1 1 50%;
  min-width: 300px;
}

        4. Apa itu box model di CSS?

        Jawaban: Box model adalah cara browser menghitung lebar/tinggi elemen: content + padding + border + margin.

        5. Bagaimana cara membuat button dengan hover effect?

        button {
  background: #007bff;
  color: white;
  border: none;
  padding: 1rem 2rem;
  border-radius: 5px;
  transition: background 0.3s;
}
button:hover {
  background: #0056b3;
}

        Latihan Live Coding

        1. Buat form login sederhana dengan HTML & CSS
        2. Buat grid gallery responsif dengan Flexbox
        3. Buat animasi hover pada card menggunakan transform dan transition

        Quiz Interview

        1. Apa perbedaan inline dan block element?
        2. Bagaimana cara membuat gambar responsif di HTML & CSS?
        3. Jelaskan fungsi z-index dan kapan digunakan!
        4. Buat contoh penggunaan media queries untuk mobile!
