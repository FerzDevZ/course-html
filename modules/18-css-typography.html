        Pelajari styling teks dan tipografi dalam CSS

        CSS Typography

        Typography adalah seni dan teknik mengatur teks agar mudah dibaca dan menarik secara visual. CSS menyediakan berbagai property untuk mengontrol tampilan teks.

        Analogi & Penjelasan

        Typography = Tulisan Tangan

        Seperti tulisan tangan yang memiliki gaya, ukuran, dan karakteristik berbeda, typography CSS mengatur bagaimana teks ditampilkan.

        Typography = Buku

        Seperti buku yang memiliki judul besar, sub-judul, dan teks body yang berbeda, typography membantu hierarki informasi.

        Font Properties

        CSS Font Properties:

        body {
  /* Font Family */
  font-family: 'Arial', sans-serif;
  font-family: 'Times New Roman', serif;
  font-family: 'Courier New', monospace;
  
  /* Font Size */
  font-size: 16px;
  font-size: 1.2rem;
  font-size: 120%;
  
  /* Font Weight */
  font-weight: normal;
  font-weight: bold;
  font-weight: 400;
  font-weight: 700;
  
  /* Font Style */
  font-style: normal;
  font-style: italic;
  font-style: oblique;
}

        Text Properties

        CSS Text Properties:

        h1 {
  /* Text Alignment */
  text-align: left;
  text-align: center;
  text-align: right;
  text-align: justify;
  
  /* Text Decoration */
  text-decoration: none;
  text-decoration: underline;
  text-decoration: line-through;
  text-decoration: overline;
  
  /* Text Transform */
  text-transform: none;
  text-transform: uppercase;
  text-transform: lowercase;
  text-transform: capitalize;
  
  /* Line Height */
  line-height: 1.5;
  line-height: 24px;
  
  /* Letter Spacing */
  letter-spacing: 2px;
  letter-spacing: -1px;
}

        Google Fonts

        Google Fonts menyediakan ribuan font gratis yang bisa digunakan di website.

        Cara Menggunakan Google Fonts:

        <!-- Di HTML head -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap" rel="stylesheet">

/* Di CSS */
body {
  font-family: 'Roboto', sans-serif;
  font-weight: 300;
}

h1 {
  font-family: 'Roboto', sans-serif;
  font-weight: 700;
}

        Latihan Praktis

        1. Buat hierarki typography dengan heading yang berbeda
        2. Eksperimen dengan berbagai font dari Google Fonts
        3. Buat text styling dengan alignment dan decoration
        4. Praktikkan responsive typography dengan rem/em
