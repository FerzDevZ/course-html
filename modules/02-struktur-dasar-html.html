Modul 2: Struktur Dasar HTML

Struktur Dasar Dokumen HTML

Setiap halaman web HTML memiliki struktur dasar yang wajib diikuti agar dapat dikenali browser dengan benar. Berikut adalah bagian-bagian utamanya:

  • <!DOCTYPE html>: Mendeklarasikan tipe dokumen HTML5.
  • <html>: Elemen root, membungkus seluruh isi dokumen.
  • <head>: Berisi metadata, judul, link CSS, dsb.
  • <body>: Berisi seluruh konten yang tampil di browser.

Contoh Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo, dunia!</h1>
    <p>Ini adalah halaman HTML pertama saya.</p>
  </body>
</html>

Penjelasan Setiap Bagian

  • <!DOCTYPE html>: Wajib di baris pertama, memberitahu browser bahwa ini dokumen HTML5.
  • <html lang="id">: Tag pembuka HTML, lang="id" artinya bahasa Indonesia.
  • <head>: Berisi info tentang dokumen (tidak tampil di halaman), seperti judul, meta, link CSS/JS.
  • <body>: Semua konten yang akan tampil di browser (teks, gambar, dsb).

Ilustrasi Struktur HTML

Latihan

  1. Buat file struktur.html di text editor Anda.
  2. Tulis struktur dasar HTML seperti contoh di atas.
  3. Ubah judul halaman dan isi <body> sesuai keinginan Anda.
  4. Buka di browser, pastikan tampil dengan benar.
