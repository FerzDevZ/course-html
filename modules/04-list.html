Modul 4: List (Daftar)

Jenis List di HTML

  • Ordered List (<ol>): Daftar bernomor.
  • Unordered List (<ul>): Daftar berpoin/bulet.
  • Description List (<dl>): Daftar istilah & deskripsi.

Contoh Ordered List

<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>

Hasilnya:

  1. HTML
  2. CSS
  3. JavaScript

Contoh Unordered List

<ul>
  <li>Apel</li>
  <li>Jeruk</li>
  <li>Mangga</li>
</ul>

Hasilnya:

  • Apel
  • Jeruk
  • Mangga

Contoh Description List

<dl>
  <dt>HTML</dt>
  <dd>Bahasa markup untuk web</dd>
  <dt>CSS</dt>
  <dd>Bahasa style untuk web</dd>
</dl>

Hasilnya:

HTML
Bahasa markup untuk web
CSS
Bahasa style untuk web

Ilustrasi List

Latihan

  1. Buat file list.html.
  2. Buat satu ordered list, satu unordered list, dan satu description list sesuai contoh.
  3. Ubah isi list sesuai minat Anda (misal: daftar hobi, makanan favorit, dsb).
  4. Buka di browser dan amati perbedaannya.

Quiz

  1. Apa perbedaan <ol> dan <ul>?
  2. Kapan sebaiknya menggunakan description list?
  3. Bolehkah list di-nest (ditumpuk) di dalam list lain?
