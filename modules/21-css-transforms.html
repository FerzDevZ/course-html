        Modul 21: CSS Transforms

        Pelajari transformasi elemen dengan CSS

        CSS Transforms

        CSS Transforms memungkinkan kita mengubah bentuk, ukuran, dan posisi elemen tanpa mempengaruhi layout dokumen.

        Analogi & Penjelasan

        Transforms = Sihir

        Seperti sihir yang mengubah bentuk benda, CSS transforms mengubah tampilan elemen tanpa mengubah struktur aslinya.

        Transforms = Kamera

        Seperti kamera yang bisa zoom, rotate, dan move, CSS transforms memungkinkan manipulasi visual elemen.

        Translate

        Translate memindahkan elemen dari posisi aslinya tanpa mempengaruhi layout.

        CSS Translate:

        .element {
  /* Translate X (horizontal) */
  transform: translateX(50px);
  transform: translateX(-20px);
  
  /* Translate Y (vertical) */
  transform: translateY(30px);
  transform: translateY(-10px);
  
  /* Translate X dan Y */
  transform: translate(50px, 30px);
  transform: translate(-20px, -10px);
  
  /* Percentage values */
  transform: translateX(50%);
  transform: translateY(100%);
}

        Rotate

        Rotate memutar elemen berdasarkan sudut tertentu.

        CSS Rotate:

        .element {
  /* Rotate dengan derajat */
  transform: rotate(45deg);
  transform: rotate(90deg);
  transform: rotate(180deg);
  transform: rotate(360deg);
  
  /* Rotate negatif (counter-clockwise) */
  transform: rotate(-45deg);
  
  /* Rotate dengan radian */
  transform: rotate(1.57rad);
  
  /* Rotate dengan gradian */
  transform: rotate(100grad);
}

        Scale

        Scale mengubah ukuran elemen (memperbesar atau memperkecil).

        CSS Scale:

        .element {
  /* Scale X (horizontal) */
  transform: scaleX(2);    /* 2x lebih lebar */
  transform: scaleX(0.5);  /* 0.5x lebih sempit */
  
  /* Scale Y (vertical) */
  transform: scaleY(1.5);  /* 1.5x lebih tinggi */
  transform: scaleY(0.8);  /* 0.8x lebih pendek */
  
  /* Scale X dan Y */
  transform: scale(2, 1.5);    /* 2x lebar, 1.5x tinggi */
  transform: scale(0.5, 0.8);  /* 0.5x lebar, 0.8x tinggi */
  
  /* Scale uniform */
  transform: scale(2);     /* 2x lebih besar */
  transform: scale(0.5);   /* 0.5x lebih kecil */
}

        Skew

        Skew membuat elemen miring atau condong.

        CSS Skew:

        .element {
  /* Skew X (horizontal) */
  transform: skewX(20deg);
  transform: skewX(-15deg);
  
  /* Skew Y (vertical) */
  transform: skewY(10deg);
  transform: skewY(-5deg);
  
  /* Skew X dan Y */
  transform: skew(20deg, 10deg);
  transform: skew(-15deg, -5deg);
}

        Multiple Transforms

        Kita bisa menggabungkan beberapa transform dalam satu property.

        Multiple Transforms:

        .element {
  /* Multiple transforms */
  transform: translateX(50px) rotate(45deg) scale(1.2);
  transform: translate(20px, 30px) rotate(-30deg) skewX(10deg);
  
  /* Transform origin */
  transform-origin: center;
  transform-origin: top left;
  transform-origin: 50% 50%;
}

        Latihan Praktis

        1. Buat hover effects dengan translate dan scale
        2. Eksperimen dengan rotate untuk membuat loading spinner
        3. Buat card flip effect dengan rotateY
        4. Praktikkan transform origin untuk animasi yang lebih natural
