Modul 6: Gambar & Multimedia

Menampilkan Gambar di HTML

Gunakan tag <img> untuk menampilkan gambar. Sumber gambar bisa dari internet atau lokal.

<img src="https://images.unsplash.com/photo-1506744038136-46273834b3fb" alt="Pemandangan Alam" width="300">

Penjelasan Atribut <img>

  • src: Alamat gambar (URL atau path lokal)
  • alt: Teks alternatif (untuk aksesibilitas & SEO)
  • width/height: Ukuran gambar (opsional)

Menampilkan Video

Gunakan tag <video> untuk menampilkan video. Bisa dari file lokal atau link internet.

<video src="https://www.w3schools.com/html/mov_bbb.mp4" controls width="320"></video>

Menampilkan Audio

<audio src="https://www.w3schools.com/html/horse.mp3" controls></audio>

Tag Multimedia Lainnya

  • <figure> & <figcaption>: Untuk gambar dengan caption.
  • <iframe>: Untuk embed video YouTube, peta, dsb.
  • <embed> & <object>: Untuk file PDF, animasi, dsb.

Ilustrasi Multimedia

Latihan

  1. Buat file gambar-multimedia.html.
  2. Tambahkan gambar dari internet dan lokal (jika ada).
  3. Tambahkan video dan audio dari internet.
  4. Gunakan <figure> dan <figcaption> untuk gambar dengan caption.
