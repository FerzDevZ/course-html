        Modul 19: CSS Colors

        Pelajari penggunaan warna dalam CSS

        CSS Colors

        CSS menyediakan berbagai cara untuk mendefinisikan warna, dari nama warna sederhana hingga nilai hexadecimal dan RGB yang kompleks.

        Analogi & Penjelasan

        Colors = Cat

        Seperti cat yang memiliki berbagai warna dan cara mencampur, CSS colors memiliki berbagai format dan cara mendefinisikan warna.

        Colors = Pelangi

        Seperti pelangi yang memiliki spektrum warna, CSS colors memungkinkan kita menggunakan jutaan kombinasi warna.

        Color Values

        Berbagai Format Warna:

        /* Named Colors */
.element {
  color: red;
  color: blue;
  color: green;
  color: transparent;
}

/* Hexadecimal */
.element {
  color: #ff0000; /* Red */
  color: #00ff00; /* Green */
  color: #0000ff; /* Blue */
  color: #fff;    /* White */
  color: #000;    /* Black */
}

/* RGB */
.element {
  color: rgb(255, 0, 0);     /* Red */
  color: rgb(0, 255, 0);     /* Green */
  color: rgb(0, 0, 255);     /* Blue */
  color: rgb(255, 255, 255); /* White */
}

/* RGBA (with opacity) */
.element {
  color: rgba(255, 0, 0, 0.5); /* Semi-transparent red */
  background-color: rgba(0, 0, 0, 0.1); /* Semi-transparent black */
}

/* HSL */
.element {
  color: hsl(0, 100%, 50%);    /* Red */
  color: hsl(120, 100%, 50%);  /* Green */
  color: hsl(240, 100%, 50%);  /* Blue */
}

        CSS Gradients

        Gradients memungkinkan transisi halus antara dua atau lebih warna.

        Linear Gradient:

        /* Linear Gradient */
.gradient {
  background: linear-gradient(to right, red, blue);
  background: linear-gradient(45deg, red, blue);
  background: linear-gradient(to bottom, red, yellow, green);
}

/* Radial Gradient */
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  background: radial-gradient(ellipse, red, blue);
}

        Color Theory

        Pemahaman dasar teori warna membantu dalam memilih kombinasi warna yang harmonis.

        Color Wheel

        Primary Colors

        Red, Blue, Yellow - warna dasar

        Secondary Colors

        Green, Orange, Purple - campuran primary

        Complementary

        Warna yang berlawanan di color wheel

        Analogous

        Warna yang berdekatan di color wheel

        Latihan Praktis

        1. Eksperimen dengan berbagai format warna (hex, rgb, hsl)
        2. Buat gradient linear dan radial
        3. Praktikkan color theory dengan complementary colors
        4. Buat color scheme untuk website
