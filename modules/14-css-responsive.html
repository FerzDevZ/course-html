        Modul 14: CSS Responsive Design

        Buat website yang responsif untuk semua perangkat

        Apa itu Responsive Design?

        Responsive Design adalah pendekatan desain web yang membuat halaman web terlihat dan berfungsi dengan baik di berbagai ukuran layar, dari desktop hingga mobile.

        Analogi & Penjelasan

        Responsive = Air

        Seperti air yang mengikuti bentuk wadahnya, website responsif akan menyesuaikan dengan ukuran layar perangkat yang digunakan.

        Responsive = Pakaian

        Website responsif seperti pakaian yang bisa dipakai oleh orang dengan ukuran berbeda. Ia akan menyesuaikan dengan "ukuran" layar.

        Media Queries

        Media Queries adalah teknik CSS untuk menerapkan style yang berbeda berdasarkan karakteristik perangkat, terutama lebar layar.

        Contoh Media Queries:

        /* Desktop */
.container {
  max-width: 1200px;
  margin: 0 auto;
}

/* Tablet */
@media (max-width: 768px) {
  .container {
    max-width: 100%;
    padding: 0 20px;
  }
}

/* Mobile */
@media (max-width: 480px) {
  .container {
    padding: 0 10px;
  }
  
  h1 {
    font-size: 24px;
  }
}

        Mobile-First Approach

        Mobile-First adalah pendekatan desain yang dimulai dengan desain untuk mobile terlebih dahulu, kemudian berkembang ke desktop.

        Mobile-First CSS:

        /* Mobile First */
.card {
  width: 100%;
  margin-bottom: 20px;
}

/* Tablet */
@media (min-width: 768px) {
  .card {
    width: 48%;
    float: left;
    margin-right: 2%;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .card {
    width: 31%;
    margin-right: 2%;
  }
}

        Viewport Meta Tag

        Viewport meta tag sangat penting untuk responsive design di mobile device.

        Viewport Meta Tag:

        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        Penjelasan:

        • width=device-width - Lebar viewport sama dengan lebar perangkat
        • initial-scale=1.0 - Zoom awal 100%
        • user-scalable=no - Mencegah user zoom (opsional)

        Responsive Images

        Gambar juga harus responsif agar tidak melebihi container dan tetap terlihat baik di semua perangkat.

        CSS untuk Responsive Images:

        img {
  max-width: 100%;
  height: auto;
  display: block;
}

.responsive-image {
  width: 100%;
  height: auto;
  object-fit: cover;
}

        Latihan Praktis

        1. Buat layout yang responsif dengan media queries
        2. Coba pendekatan mobile-first untuk desain
        3. Buat navigation yang berubah dari horizontal ke hamburger menu di mobile
        4. Test website di berbagai ukuran layar
