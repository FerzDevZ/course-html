Modul 7: Table

Membuat Tabel di HTML

Tabel digunakan untuk menampilkan data dalam baris dan kolom. Tag utama: <table>, <tr> (baris), <th> (header), <td> (data).

Contoh Tabel Sederhana

<table border="1">
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Andi</td>
    <td>20</td>
  </tr>
  <tr>
    <td>Budi</td>
    <td>22</td>
  </tr>
</table>

Hasilnya:

NamaUmur
Andi20
Budi22

Struktur Tabel Lanjutan

  • <thead>: Kepala tabel
  • <tbody>: Isi tabel
  • <tfoot>: Kaki tabel
  • colspan & rowspan: Gabung kolom/baris

Contoh Gabung Kolom

<td colspan="2">Gabung 2 kolom</td>

Ilustrasi Tabel

Latihan

  1. Buat file table.html.
  2. Buat tabel data siswa (nama, umur, nilai) minimal 3 baris.
  3. Coba gunakan colspan dan rowspan.
  4. Tambahkan <thead>, <tbody>, dan <tfoot> jika perlu.
