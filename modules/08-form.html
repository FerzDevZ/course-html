Modul 8: Formulir (Form)

Membuat Formulir di HTML

Formulir digunakan untuk mengumpulkan data dari pengguna. Tag utama: <form>, <input>, <textarea>, <select>, <button>, <label>.

Contoh Formulir Sederhana

<form action="#" method="post">
  Nama: <input type="text" name="nama"><br>
  Email: <input type="email" name="email"><br>
  Pesan: <textarea name="pesan"></textarea><br>
  <button type="submit">Kirim</button>
</form>

Hasilnya:

Nama:
Email:
Pesan:

Jenis Input pada Form

  • text: Input teks biasa
  • email: Input email (validasi otomatis)
  • password: Input sandi
  • radio: Pilihan satu dari beberapa
  • checkbox: Pilihan jamak
  • file: Upload file
  • date, number, range, dsb.

Contoh Input Radio & Checkbox

<input type="radio" name="gender" value="pria"> Pria
<input type="radio" name="gender" value="wanita"> Wanita
<br>
<input type="checkbox" name="hobi" value="musik"> Musik
<input type="checkbox" name="hobi" value="olahraga"> Olahraga

Label & Fieldset

<label for="nama">Nama Lengkap</label>
<input id="nama" name="nama">

<fieldset>
  <legend>Data Diri</legend>
  ...
</fieldset>

Ilustrasi Formulir

Latihan

  1. Buat file form.html.
  2. Buat form pendaftaran sederhana (nama, email, password, gender, hobi, pesan).
  3. Gunakan <label> dan <fieldset> untuk aksesibilitas.
  4. Coba submit form dan amati hasilnya.
